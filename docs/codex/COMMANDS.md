# Commands

Comandos versionados y ejecutables:

| Comando | Uso |
| --- | --- |
| `pnpm install` | instalar dependencias |
| `pnpm dev` | desarrollo |
| `pnpm start` | correr build existente |
| `pnpm lint` | lint |
| `pnpm build` | build |

No existe comando versionado para: tests, Playwright, deploy, migraciones locales.
