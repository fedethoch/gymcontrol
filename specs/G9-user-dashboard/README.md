# G9 - User Dashboard

Este grupo implementa la experiencia del usuario autenticado dentro de su dashboard.

## Objetivo del grupo

Permitir guardar varias rutinas, asignarles nombres propios y consultarlas desde la cuenta del usuario.

## Orden recomendado

1. `01-implementar-accion-para-guardar-rutina-en-cuenta.md`
2. `02-permitir-que-el-usuario-tenga-mas-de-una-rutina-guardada.md`
3. `03-definir-representacion-entre-rutina-plantilla-y-rutina-guardada.md`
4. `04-permitir-asignar-nombre-propio-a-cada-rutina-guardada.md`
5. `05-permitir-editar-nombre-desde-el-dashboard.md`
6. `06-crear-dashboard-de-usuario-con-listado-de-rutinas-guardadas.md`
7. `07-permitir-ver-detalle-de-cada-rutina-guardada.md`
8. `08-mantener-acceso-al-modal-de-detalle-de-ejercicio.md`

## Resultado esperado

Tener un dashboard de usuario util, simple y alineado con el flujo principal del MVP.
