# G8 - Routine Catalog

Este grupo implementa el catalogo de rutinas disponible para los usuarios.

## Objetivo del grupo

Permitir explorar rutinas, ver su detalle semanal y preparar el punto de guardado en cuenta del usuario.

## Orden recomendado

1. `01-crear-vista-catalogo-de-rutinas-disponibles.md`
2. `02-mostrar-informacion-minima-util-de-cada-rutina.md`
3. `03-permitir-entrar-al-detalle-de-una-rutina.md`
4. `04-mostrar-estructura-semanal-de-la-rutina-elegida.md`
5. `05-hacer-clickeables-los-ejercicios-para-su-detalle.md`
6. `06-preparar-punto-de-accion-para-guardar-rutina.md`

## Resultado esperado

Tener un catalogo claro que permita explorar rutinas y avanzar al guardado en cuenta.
