# G4 - MVP Setup

Este grupo prepara la base tecnica del proyecto despues de cerrar arquitectura, base de datos y estrategia de trabajo.

## Objetivo del grupo

Dejar una base tecnica limpia, navegable y lista para recibir funcionalidades del MVP.

## Orden recomendado

1. `01-confirmar-stack-final-y-dependencias-necesarias.md`
2. `02-instalar-y-configurar-librerias-imprescindibles.md`
3. `03-limpiar-el-boilerplate-inicial.md`
4. `04-crear-estructura-de-carpetas-pactada.md`
5. `05-configurar-estilos-base-y-ui-compartida.md`
6. `06-configurar-acceso-a-base-de-datos-y-entorno.md`
7. `07-crear-layout-base-y-navegacion-inicial.md`
8. `08-dejar-esqueleto-navegable-de-las-vistas-principales.md`

## Resultado esperado

Tener el proyecto listo para avanzar con auth, admin y experiencia de usuario sin reorganizaciones fuertes.
