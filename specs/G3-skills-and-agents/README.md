# G3 - Skills And Agents

Este grupo define que skills o agentes realmente hacen falta para trabajar el proyecto con eficiencia.

## Objetivo del grupo

Mantener el set de skills y agentes al minimo necesario, evitando solapamientos y gasto innecesario de tokens.

Tener `docs/SKILLS_AND_AGENTS.md` con reglas de uso claras y optimizadas.
