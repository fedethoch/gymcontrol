# G4.5 - Frontend Base And Experience

Este grupo redefine la base visual y de experiencia del MVP antes de avanzar con auth y logica funcional.

## Objetivo del grupo

Dejar una experiencia frontend clara, moderna y coherente para usuario y admin, apoyada en un panel lateral izquierdo, vistas principales bien separadas, una lectura semanal del entrenamiento y un patron consistente para detalle de rutina y modal de ejercicio.
