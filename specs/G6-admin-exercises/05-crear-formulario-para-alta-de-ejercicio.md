# Crear formulario para alta de ejercicio: permitir la carga administrativa del recurso

**Grupo:** G6 - Ejercicios admin  
**Orden:** 5  
**Esfuerzo:** Medio  
**Modelo recomendado:** gpt 5.4 mini

## Objetivo

Crear el formulario de alta de ejercicio dentro del admin dashboard para que el administrador pueda registrar nuevos ejercicios en el sistema.

## Estado

Pendiente.

## Archivos

- vistas del admin dashboard
- acciones de ejercicio
- validaciones de ejercicio
- componentes de formulario

## Pasos

1. Crear la vista o seccion de alta de ejercicio.
2. Construir el formulario con los campos requeridos.
3. Integrar validaciones y accion de guardado.
4. Confirmar feedback minimo tras la creacion.

## Criterios de aceptacion

- El admin puede crear un ejercicio con nombre, descripcion e imagen.
- El formulario usa las validaciones definidas.
- El ejercicio queda persistido correctamente.

## Resolucion

Completar al ejecutar este paso.

## Validacion

- Verificar que el flujo de alta funcione de punta a punta.
- Verificar que el resultado aparezca luego en el listado de ejercicios.
