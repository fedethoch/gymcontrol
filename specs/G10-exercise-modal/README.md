# G10 - Exercise Modal

Este grupo implementa el detalle emergente de ejercicio como componente transversal del MVP.

## Objetivo del grupo

Unificar la experiencia de consulta de ejercicio sin salir de la pagina actual, con un componente reutilizable y consistente antes de cerrar catalogo y dashboard de usuario.

## Orden recomendado

1. `01-disenar-e-implementar-componente-base-de-modal.md`
2. `02-mostrar-nombre-imagen-y-descripcion-del-ejercicio.md`
3. `03-permitir-apertura-y-cierre-sin-abandonar-la-pagina.md`
4. `04-integrar-el-modal-en-catalogo-de-rutinas.md`
5. `05-integrar-el-modal-en-dashboard-de-usuario.md`
6. `06-integrar-el-modal-en-otras-vistas-del-mvp-que-muestren-ejercicios.md`
7. `07-asegurar-comportamiento-claro-en-desktop-y-mobile.md`
8. `08-revisar-accesibilidad-minima-de-foco-cierre-y-lectura.md`

## Resultado esperado

Tener un modal de ejercicio estable, reutilizable y coherente en todas las vistas relevantes.

## Dependencias operativas

- debe apoyarse en el shell y patrones definidos en `G4.5`
- conviene cerrarlo antes de integrar en serio `G8` y `G9` para evitar retrabajo
