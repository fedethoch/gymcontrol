## Objetivo

Disenar e implementar un componente base de modal o emergente reutilizable para mostrar el detalle de un ejercicio sin abandonar la pagina actual.

2. Disenar la estructura base del componente modal.

4. Evitar acoplar el componente a una sola vista o modulo.

## Criterios de aceptacion

- Existe un componente base reutilizable de modal.
- El componente puede usarse en multiples vistas del MVP.
- La implementacion es simple y mantenible.
