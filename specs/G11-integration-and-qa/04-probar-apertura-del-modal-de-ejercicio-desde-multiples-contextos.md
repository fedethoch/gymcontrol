# Probar apertura del modal de ejercicio desde multiples contextos: validar consistencia transversal

**Grupo:** G11 - Integracion y QA del MVP  
**Orden:** 4  
**Esfuerzo:** Bajo  
**Modelo recomendado:** gpt 5.4 mini

## Objetivo

Probar que el modal de detalle de ejercicio funcione correctamente desde todos los contextos relevantes del MVP.

## Estado

Pendiente.

## Archivos

- modal de ejercicio
- catalogo de rutinas
- dashboard de usuario
- otras vistas con ejercicios clickeables

## Pasos

1. Abrir el modal desde el catalogo.
2. Abrir el modal desde el dashboard de usuario.
3. Probar apertura, lectura y cierre en cada contexto.
4. Registrar inconsistencias visuales o funcionales.

## Criterios de aceptacion

- El modal funciona en todos los contextos previstos.
- El ejercicio mostrado coincide con el clickeado.
- El usuario no pierde el contexto de la vista actual.

## Resolucion

Completar al ejecutar este paso.

## Validacion

- Verificar que apertura y cierre se comporten igual en todas las vistas.
- Verificar que nombre, imagen y descripcion se muestren correctamente.

## Estado final

No iniciado.
