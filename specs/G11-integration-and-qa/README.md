# G11 - Integration And QA

Este grupo valida el MVP de punta a punta antes del cierre final.

## Objetivo del grupo

Probar flujos criticos, permisos, errores y consistencia general para detectar y corregir problemas antes del cierre del MVP.

## Orden recomendado

1. `01-probar-flujo-admin-de-creacion-de-ejercicios.md`
2. `02-probar-flujo-admin-de-creacion-de-rutinas-semanales.md`
3. `03-probar-flujo-usuario-de-exploracion-guardado-y-renombrado.md`
4. `04-probar-apertura-del-modal-de-ejercicio-desde-multiples-contextos.md`
5. `05-revisar-validaciones-mensajes-de-error-y-estados-vacios.md`
6. `06-revisar-permisos-para-evitar-acciones-admin-desde-no-admin.md`
7. `07-revisar-consistencia-visual-minima-entre-vistas.md`
8. `08-resolver-bugs-detectados-en-pruebas-finales.md`
9. `09-ejecutar-validacion-tecnica-final-del-mvp.md`

## Resultado esperado

Tener el MVP validado y en condiciones de pasar al grupo de cierre.
